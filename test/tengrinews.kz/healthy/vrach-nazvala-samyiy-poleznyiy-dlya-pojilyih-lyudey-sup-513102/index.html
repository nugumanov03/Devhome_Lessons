Врач назвала самый полезный для пожилых людей суп

Сегодня, 03:15
7

Самым полезным супом для пожилых людей считаются щи из квашеной капусты. Об этом "Москве 24" рассказала врач-диетолог Марият Мухина, передает Tengri Life.

По ее словам, такое блюдо поддержит работу желудочно-кишечного тракта, а также поможет насытить организм железом.

"У пожилых людей часто деградируют ферментные системы, слизистая становится более гладкой. Синтезируется меньше желудочного сока. Кислинка из квашеной капусты будет усиливать ферментные системы и создавать дополнительную кислотность, что помогает переваривать пищу", - пояснила доктор медицинских наук.

Квашеная капуста - ферментированный продукт, который содержит много полезных веществ. Некоторые из них сохраняются даже после термической обработки.

"В щах из квашеной капусты есть железо, некоторое количество витамина C и кальций. Последний укрепляет кости. Железо же нужно для хорошего гемоглобина, а витамин C помогает усваивать этот микроэлемент. Если щи будут еще и с мясом, то железа станет гораздо больше", - отметила Мухина.

Эксперт добавила, что пектин и пищевые волокна квашеной капусты связывают токсины и являются своеобразным энтеросорбентом, способствующим выведению из организма вредных веществ.
